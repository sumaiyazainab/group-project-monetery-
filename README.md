# Directed Evolution Analysis Web Portal
## MSc Bioinformatics Software Development Group Project 2026
The Directed Evolution Monitoring Web Portal is a computational platform designed to support the tracking, analysis, and visualisation of data generated from droplet-based microfluidics directed evolution (DE) experiments, which specifically targets DNA polymerase optimisation. The portal automates the end-to-end analysis pipeline, from experiment staging and plasmid validation through to variant activity scoring, mutation annotation, and visualisation, transforming raw experimental measurements into interpretable biological decisions within the Design–Build–Test–Learn (DBTL) cycle. The portal is intended for experimental, synthetic, and computational biologists, as well as protein engineers, working with large directed evolution variant libraries that span across multiple generations.
