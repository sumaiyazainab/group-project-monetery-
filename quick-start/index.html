    Quick Start Guide

    1. Create an Account

    1. Click Register/Login
    2. Enter your email, username, and password
    3. Submit the form

    2. Starting a New Experiment

    1. Navigate to the Experiments page using the top menu
    2. Click Create New Experiment
    3. Enter an experiment name
    4. Enter a UniProt accession ID
    5. Upload your plasmid FASTA file
    6. Specify the expected start codon (optional)
    7. Create New Experiment

    Important

    • Use a valid UniProt accession ID (e.g. P05067)
    • Only one plasmid FASTA file is allowed per experiment
    • If no start codon is selected, the system will default to considering all three: ATG, GTG, and TTG

    3. ORF Staging/Validation Phase

    The system will automatically:

    1. Retrieve the UniProt record and store the protein sequence and key domain annotations
    2. Validate the plasmid FASTA sequence
    3. Detect candidate ORFs across all six reading frames
    4. Select the best ORF by alignment against the UniProt reference protein
    5. Return the match status (match or no_match)

    4. Upload Directed Evolution Data

    Upload your directed evolution experimental dataset (TSV or JSON format). The file must contain the following required columns:

    • Plasmid_Variant_Index
    • Parent_Plasmid_Variant
    • Directed_Evolution_Generation
    • Assembled_DNA_Sequence
    • DNA_Quantification_fg
    • Protein_Quantification_pg
    • Control

    5. View Results

    Once analysis is complete, the following outputs are generated:

    Top 10 Ranked Variant Table — Lists the highest-performing variants ranked by their Activity Score, allowing quick identification of variants with improved catalytic performance compared to the wild-type enzyme.

    Activity Distribution per Generation — Shows how Activity Scores are distributed across each generation, helping users observe whether overall activity improves, decreases, or stabilises over successive rounds of selection.

    3D Activity Landscape — Maps variants in sequence space and displays their Activity Scores as a three-dimensional surface, helping users explore how sequence diversity relates to functional performance.

    Downloadable PDF Report — The complete analysis report, including all tables and visualisations, can be downloaded as a PDF for record keeping or further analysis.
