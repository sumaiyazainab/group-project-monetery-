    System Requirements

    1. Software Requirements

    • Python ≥ 3.9
    • Active internet connection (required for UniProt retrieval)

    The system relies on several Python libraries for data processing, sequence analysis, and visualisation. These dependencies are listed in the requirements.txt file and must be installed before running the application.

    Library Purpose
    Flask Web application framework for routing and the user interface
    SQLAlchemy Object-relational mapping for database interaction
    Pandas Data parsing and quality control for experimental datasets
    Biopython Biological sequence handling, translation, and alignment
    Requests Retrieval of protein records from the UniProt API
    NumPy Numerical operations used during analysis and visualisation
    Matplotlib Static plotting for activity distribution graphs
    Plotly Interactive 3D visualisations of the activity landscape
    Scikit-learn PCA dimensionality reduction and K-Nearest Neighbour regression

    2. Installation

    Step 1. Clone the repository: 

    git clone https://github.com/sumaiyazainab/group-project-monetery-.git

    Step 2. Install dependencies: 

    pip install -r requirements.txt

    Step 3. Run the application: 

    flask run

    Step 4. Open your browser and navigate to: 

    http://127.0.0.1:5000

    3. Supported Data Formats

    Input Format Extension
    Plasmid sequence FASTA .fasta or .fa
    Experimental data TSV .tsv
    Experimental data JSON .json
