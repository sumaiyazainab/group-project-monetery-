    Troubleshooting Guide

    UniProt and Staging Issues

    Issue Error Message Troubleshooting Steps
    UniProt accession not found ValueError: The UniProt accession 'X' does not exist... (HTTP 404) Confirm the accession ID is correct (e.g. P05067). Search for the target protein on UniProt manually and copy the accession ID.
    UniProt accession format invalid ValueError: The UniProt ID format you entered is invalid... (HTTP 400) Ensure you are not using a gene name instead of an accession. Use canonical accession format (e.g. Q9Y2T1) and remove any extra spaces.
    UniProt server error RuntimeError: UniProt server error. Please try again later. (5xx) Retry later. Verify internet access and check UniProt REST availability.
    Request timed out RuntimeError: We could not connect to UniProt in time... Check your network connectivity and try again.
    Network/connection error RuntimeError: An unexpected network error occurred... Confirm your device has internet access. Disable any VPN or proxy if present.
    UniProt record missing sequence RuntimeError: The UniProt record is missing a protein sequence. Verify the accession ID and confirm a protein sequence is displayed on the UniProt website. If the issue persists, the UniProt REST API structure may have changed — report the issue.

    FASTA and ORF Issues

    Issue Error Message Troubleshooting Steps
    FASTA contains multiple sequences ValueError: There is more than one plasmid sequence in the FASTA file... Ensure the uploaded FASTA contains exactly one record (one > header line and one continuous sequence).
    FASTA contains zero sequences ValueError: No plasmid sequence was found in the FASTA file... Ensure the file is a valid FASTA, starts with a > header, and contains sequence lines.
    Invalid DNA characters ValueError: Invalid characters found in DNA sequence: [...] Remove any non-ACGTN characters (e.g. numbers, whitespace, or IUPAC ambiguity codes beyond N). Convert sequence to A/C/G/T/N only.
    No ORFs returned No ORFs returned Verify the plasmid contains a coding region and double-check the start codon selection.
    No ORF matches UniProt Status = no_match_found Confirm the correct UniProt accession ID and plasmid have been provided and are compatible.
